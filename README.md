## 기능

선문대학교 학생들이 자주 사용하는 웹사이트를 쉽게 접근 할 수 있습니다.

현재 제공하는 사이트는 다음과 같습니다.
- 학교 메인 페이지
- 선문대학교 포털
- E-강의동
- 수강신청 페이지
- 교내 번호 검색
- 학사 일정
- 셔틀버스
- 공지사항
- 선문가이드
